Interior Craft : Документация разработчика

Created by Sergey Makovkin, last modified on Jan 17, 2021

Программа Interior Craft может работать в двух режимах:

  • Режим Пользователя

  • Режим Разработчика

Режим Разработчика имеет расширенный функционал и позволяет добавлять новый контент в базу элементов программы. Работа рядового пользователя в режиме разработки крайне не рекомендуется по двум причинам: а) программа работает заметно медленнее; б) есть риск испортить базу элементов или материалов.

Для запуска программы в режиме разработчика необходимо создать ярлык для программы, запускающей ее с дополнительным параметром:

InteriorCraft.exe --dev

ВНИМАНИЕ! Если программа установлена в директорию “C:\Program Files”, то режим разработчика не будет работать корректно, т.к. доступ к Program Files у пользователя в Windows ограничен. Программу надо либо запускать в режиме Администратора, либо изначально устанавливать в другую папку, например в “С:\Interior Craft”. Если вы уже установили программу в Program Files, то можно просто скопировать ее в “C:\Interior Craft” и запускать оттуда. Это будет ваша “рабочая” копия программы для экспериментов.
