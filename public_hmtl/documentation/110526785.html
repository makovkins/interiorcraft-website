Interior Craft : Системные требования

Created by Sergey Makovkin, last modified on May 25, 2021

Минимальные системные требования для работы программы

  • Процессор Intel Core i5 8600K или AMD Ryzen 5 2600

  • 8 GB RAM

  • Монитор с разрешением экрана не менее 1920x1080

  • Windows 7 SP3 64-bit

  • DirectX 11 совместимая видео карта

  • Наличие интернет соединения для активации Программного Продукта

Рекомендуемые требования

  • Процессор Intel Core i7 или AMD Ryzen 7

  • 8 GB RAM

  • Монитор с разрешением экрана не менее 1920x1080

  • Windows 10 64-bit (версии 1903 или более новая)

  • Видео карта NVIDIA RTX 2060

  • Наличие интернет соединения для активации Программного Продукта
