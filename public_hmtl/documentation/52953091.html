1. Проекции

Просматривать и редактировать помещение и интерьер возможно в различных проекциях (видах).
Выбор проекции осуществляется на панели “Интерьер”:

Основная работа ведется в Перспективной проекции и проекции Сверху.

2. Режим работы с выбранной стеной

Часто при создании интерьера возникает необходимость работать исключительно с одной стеной помещения, отключив все остальные.

Выбрать активную стену, отключив прочие, можно в выпадающем меню Стена на панели Интерьер:

Режим Стена часто используется для создания чертежей, расстановки размерных линий. Например, чертеж шкафа или гардеробной системы, находящейся на одной стене, удобно создавать в режиме стены.

При включении режима стены, по умолчанию используется вид Спереди. Вид можно менять на любой другую кнопкой смены проекции.

Отключение режима cтены и возврат к стандартному просмотру интерьера осуществляется повторным нажатием на кнопку Стена.

3. Настройка чувствительности управления камерой

Экспериментальная функция для продвинутых пользователей

Чтобы изменить чувствительность управления камерой необходимо в файл Game.ini, находящийся в папке %LOCALAPPDATA%\InteriorCraft\Saved\Config\WindowsNoEditor, добавить следующие строки, заменив значение по умолчанию 1.0 на требуемое:

[/Script/ApartmentCore.BasePawn]
TranslateSensetivity=1.0
RotateSensetivity=1.0
ZoomSensetivity=1.0
